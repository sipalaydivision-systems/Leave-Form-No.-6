Migrate Leave Cards & CTO Records from Excel

Upload employee Excel files from the old manual system. Each file should be one employee's data (filename = "LASTNAME, FIRSTNAME.xlsx").

Non-teaching: Leave Card (Sheet 1) + CTO Card (Sheet 2, if present) will both be migrated. Teaching: CTO records are expected; an empty Leave Card placeholder will be created if no Leave Card data exists.

Matching uses first name + last name (from filename) as the primary key, with employee number (from Leave Card) as fallback.

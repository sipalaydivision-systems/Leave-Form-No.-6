📊 Migrate Leave Cards from Excel

Upload the Excel leave card files from the old manual system. Each file should be one employee's leave card (filename = "LASTNAME, FIRSTNAME.xlsx").

The system will extract the latest VL and SL balances and transaction history from each file. A safety backup is created automatically before importing.

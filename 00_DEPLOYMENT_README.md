## 🚀 HOW TO GET LIVE TODAY (15-45 minutes)

### STEP 1: Read the Quick Start (5 min)
Open: **QUICK_START_DEPLOYMENT.md**

### STEP 2: Prepare Locally (5 min)
```bash
npm install
npm start  # Test it works
Ctrl+C     # Stop it
```

### STEP 3: Push to GitHub (5 min)
```bash
git init
git add .
git commit -m "Ready for production"
git remote add origin https://github.com/YOUR_USERNAME/leave-form.git
git push -u origin main
```

### STEP 4: Deploy to Railway (5 min)
1. Go to https://railway.app
2. Login with GitHub
3. Deploy from your repository
4. Railway auto-detects Node.js app

### STEP 5: Configure (3 min)
Set these environment variables in Railway:
- `NODE_ENV` = `production`
- `PORT` = `3000`
- `PRODUCTION_DOMAIN` = `https://[your-railway-url]`
- `MAILERSEND_API_KEY` = `[your key]`
- `MAILERSEND_SENDER_EMAIL` = `[your email]`

### STEP 6: Test (5 min)
- Visit your Railway URL
- Test all portals
- Submit a form
- **You're live!** 🎉

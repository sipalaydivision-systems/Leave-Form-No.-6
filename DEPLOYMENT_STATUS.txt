🚀 FASTEST PATH TO LIVE (15-45 MINUTES)

1. LOCAL SETUP (5 minutes)
   $ npm install
   $ npm start
   [Test works? Ctrl+C]

2. GITHUB PUSH (5 minutes)
   $ git init
   $ git add .
   $ git commit -m "Production ready"
   $ git push to https://github.com/YOUR_USERNAME/leave-form.git

3. DEPLOY TO RAILWAY (5 minutes)
   → Go to https://railway.app
   → Connect GitHub
   → Select repository
   → Click Deploy
   [Wait 2-3 minutes]

4. CONFIGURE (3 minutes)
   → Set environment variables:
      NODE_ENV = production
      PRODUCTION_DOMAIN = [your Railway URL]
      MAILERSEND_API_KEY = [your key]
      MAILERSEND_SENDER_EMAIL = [your email]
   → Deploy again

5. TEST & LAUNCH (5 minutes)
   → Visit your app URL
   → Test all 5 portals
   → Submit a form
   → Done! ✅

TOTAL TIME: ~25-35 minutes to go LIVE!
